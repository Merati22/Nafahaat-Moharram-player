# Moharram Music Player

A music player for Nafahaat Group, built with Laravel.

**Website:** [https://nafahaatgroup.ir](https://nafahaatgroup.ir)

## Installation

```bash
git clone [your-repository-url]
cd moharram-music-player
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```

## Requirements

- PHP >= 8.0
- Composer
- MySQL/PostgreSQL
- Laravel 10.x
